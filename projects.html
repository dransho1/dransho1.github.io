Face-Swapping in Real Time

Computer Vision, Spring 2016

Most recently, I worked with a partner to implement a facial swapping program in Python/OpenCV. Faces are detected using OpenCV's feature-based cascade classifiers, running on Viola-Jones facial recognition. We blended using Laplacian pyramid blending, blending low-frequency features over a large distance and high-frequency features over shorter distances. Our results were best when the images and video were well aligned and of similar depth. Thanks to Professor Zucker for the help and direction :)

In an extension, we added an optional eye-swapping feature with some interesting (and sometimes monstrous) results.

Speech Recognition Using Fast Fourier Transform

Digital Signal Processing, Spring 2016

In DSP I worked with a partner to build a speech recognition program. We developed the program in MATLAB to be able to explore classifiers and the fast fourier transform. Our program distinguished between ‘yes’ and ‘no’ audio signals by training on a set of known y/n inputs. For each word, we evaluated the power spectral density of the FFT to represent the energy content of signal in the frequency domain. We then created a feature value based on the average power spectrum distinguishing y/n. This acted as a linear classifier between the two words. By comparing the PSD of unknown input to the known feature value, our program can predict if the unknown word is either 'yes' or 'no.' This type of technology forms the very basics of automatic speech recognition. Based on a project from the University of Portland.

LED Color Light Organ

Electronic Circuit Applications, Fall 2015

For this project, my partner and I built an LED light organ that when music was being played, three sets of LEDs would react to three different frequency ranges, flashing to the beat of the music. In order to accomplish this we used three broadband band-pass filters where we set the frequency ranges using Sallen-Key topology, coupled with an analog signal microphone, peak detectors, and comparators to control the LEDs. We designed each stage of the circuit, and mounted the design on a breadboard.
